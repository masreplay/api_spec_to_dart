# Release Notes

## 4.1.0 - 2025-12-03

### New Features

- **Global Imports Configuration**: You can now specify global imports that will be automatically added to all generated API client and model files. This makes it easier to include common dependencies like custom converters or utility packages across your generated code without manually editing each file.

### Improvements

- **Better Code Organization**: Improved internal code structure by removing unused code and simplifying class name extraction logic, resulting in cleaner and more maintainable generated code.

### Bug Fixes

- Fixed code generation issues by cleaning up unused parser code and improving the overall code generation process.
